发表于 2020-07-26

Docker常用命令

基础命令

列出所有docker支持的命令

docker

搜索镜像

docker search zookeeper

OFFICIAL这一列上有[OK]的为官方提供的镜像

下载镜像

docker pull zookeeper

查看所有已经下载到本地的镜像

docker images

运行容器

docker run -it zookeeper

使用-it会打开一个shell与容器进行交互。容器启动并运行后，我们可以像普通机器那样来使用它，在容器中执行命令。

显示所有的docker容器

显示所有运行中的容器：

docker ps

显示所有运行中和非运行中的容器：

docker ps -a

停止容器

docker stop container-id

进入容器中

docker exec -it container-id/container-name bash

bash要根据环境的不同进行替换

从容器中退出

如果在shell状态和容器交互，要退出，执行：

exit

保存容器状态

容器运行并更改后（比如安装了 apache 服务器），我们可以保存容器状态。这会在本地系统上保存新创建镜像。运行下面语句来提交并保存容器状态：

docker commit 85475ef774 repository/image_name

这里，commit命令会保存容器状态。85475ef774是容器的容器id，而不是镜像id。repository通常为Docker Hub上的用户名（或者新加的仓库名称）。image_name是新镜像的名称。

我们还可以使用-m和-a来添加更多信息。通过-m，我们可以留个信息说apache服务器已经安装好了，而-a可以添加作者名称。像这样：

docker commit -m "apache server installed" -a "Dan Daniels" 85475ef774 daniels_dan/Cent_container
